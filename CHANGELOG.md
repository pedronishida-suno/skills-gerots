# Changelog — Super Gerots Skills

Todas as mudanças notáveis neste projeto serão documentadas aqui.
Formato segue [Keep a Changelog](https://keepachangelog.com/).

## [v2.0.0] - 2026-04-14
### Adicionado
- Skill `suno-research`: teses de investimento, valuation DCF, comparativo de ativos, síntese para distribuição
- Skill `suno-assets`: relatório gerencial, monitoramento de risco, material para cotistas, simulação de cenários
- Skill `suno-juridico`: validação de materiais, enforcement Chinese Wall, revisão de contratos, mapeamento regulatório
- Skill `suno-gerrot-navigator`: estratificação L1→L2→L3, comandos /estratificar /aprender /finalizar, guardrails anti-dispersão
- `MEMORIA.md`: alertas CORE 2025, Leads Bear, árvore hierárquica Gerrot/OKR, regra Pareto 80/20, threshold bonificação 3.5
### Alterado
- `install.js`: migrado de curl+GITHUB_TOKEN para gh CLI, adicionado comando `setup`, status mostra skills instaladas
- `MEMORIA.md`: renumeração de seções (2→4, 3→5, etc.) para acomodar novas seções 2 e 3

## [v1.0.0] - 2026-04-14
### Adicionado
- Skill `suno-core`: output obrigatório, integração Google Sheets, bonificação
- Skill `suno-fpa`: variance analysis, forecast, DRE, material de resultado
- Skill `suno-marketing`: copy, funil, CAC/LTV, briefing de conteúdo
- Skill `suno-consultoria-wealth`: Journey Book, suitability, carteira, onboarding
- Skill `suno-feedback`: coleta, classificação, geração automática de patches
- `MEMORIA.md`: contexto Suno S.A., glossário, regras, perfis de usuário
